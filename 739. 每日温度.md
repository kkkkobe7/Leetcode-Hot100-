# 739. 每日温度

题干简化含义：对于一个数组，找到每一个元素后面第一个比其大的元素的相对位置。

由于需要找后面比前面大的元素，所以需要从后向前遍历。

对于数组`nums[]`中元素`nums[k]`，如果有`nums[k-1] >= nums[k]`，那么则不需要存储`nums[k]`，因为`nums[k-1]`相对位置更靠前。这就是单调栈的经典场景，此题中单调栈内满足单调递增。

具体情况如下：

1. 如果当前温度高于栈顶，则弹出栈顶，直至找到比当前温度高（由于栈后进先出的性质，第一个找到的更高温度值一定是对应位置的答案）的或者栈为空。
2. 将当前元素加入栈内（此时一定满足单调递增的性质）
3. 由于题目需要计算相对位置，存储在栈内的是数组下标

```c++
class Solution {
public:
    vector<int> dailyTemperatures(vector<int>& temperatures) {
        int n = temperatures.size();
        vector<int> ans(n, 0);
        // 找到每一个元素后面第一个比其大的元素位置
        stack<int> st; // 单调栈
        st.push(n - 1);
        for (int i = n - 2; i >= 0; i --)
        {
            // 如果栈非空，且当前温度比栈顶更高，则弹出栈顶。
            while(!st.empty() && temperatures[i] >= temperatures[st.top()])
            {
                st.pop();
            }
            ans[i] = (st.empty())? 0: st.top() - i;
            st.push(i);
        }
        return ans;
    }
};
```
