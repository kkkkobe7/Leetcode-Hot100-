# 96. 不同的二叉搜索树

 解题思路拆解

**1. 核心发现：根节点决定结构**
*   给定 $1 \dots n$ 这 $n$ 个连续数字，如果我们选定其中一个数字 $i$ 作为根节点：
    *   根据 BST 的性质，比 $i$ 小的数字（$1 \dots i-1$）必须全部在**左子树**。
    *   比 $i$ 大的数字（$i+1 \dots n$）必须全部在**右子树**。
*   **结论**：以 $i$ 为根的 BST 数量 = **(左侧 $i-1$ 个节点组成的 BST 数量) $\times$ (右侧 $n-i$ 个节点组成的 BST 数量)**。

**2. 定义状态 (DP State)**
*   我们用 `G(n)` 表示 $n$ 个节点能组成的独特 BST 的总数。
*   我们的目标是求 `G(n)`。

**3. 状态转移方程 (Transition)**
*   对于 $n$ 个节点，根节点可以是 $1, 2, \dots, n$ 中的任意一个。
*   如果根是 $1$：左边 0 个节点，右边 $n-1$ 个。数量 = $G(0) \times G(n-1)$。
*   如果根是 $2$：左边 1 个节点，右边 $n-2$ 个。数量 = $G(1) \times G(n-2)$。
*   ...
*   如果根是 $j$：左边 $j-1$ 个节点，右边 $n-j$ 个。数量 = $G(j-1) \times G(n-j)$。
*   **总公式**：$G(i) = \sum_{j=1}^{i} G(j-1) \times G(i-j)$。

**4. 初始状态 (Base Case)**
*   `G(0) = 1`：空树也是一种结构。
*   `G(1) = 1`：只有一个节点的树只有一种结构。



简洁版代码：

```c++
class Solution {
public:
    static const int N = 20;
    int a[N] = {1, 1};
    // 假设g[n]为n个节点存在的时候二叉搜索树个数，f[n]为以i为根的二叉搜索树个数
    // g[n] = f[1] + f[2] + ... + f[n]
    // 当i为根节点的时候，左子树节点个数为i-1个，右子树节点为n-1 f[i] = g[i-1]*g[n-1]
    // g[n] = g[0]*g[n-1] + g[1]*g[n-2] + ... + g[n-1]*g[0]
    
    int numTrees(int n) {
        if (a[n] != 0) return a[n];
        for (int i = 2; i <= n; i ++)
            for (int j = 1; j <= i; j ++)
                a[i] += a[j-1]*a[i-j];
        return a[n];
    }
};
```

注释版代码：

```c++
class Solution {
public:
    // N 定义为 20，因为题目给出 n 的范围通常很小 (n <= 19)
    static const int N = 20;
    
    // a[i] 存储 i 个节点能组成的 BST 种类数
    // 初始化 a[0] = 1, a[1] = 1
    int a[N] = {1, 1};

    int numTrees(int n) {
        // 1. 记忆化检查：如果已经计算过 n 的结果，直接返回
        // 注意：这里 a[n] != 0 的前提是 n > 1 且逻辑上不会出现 0 种
        if (n <= 1) return 1;
        if (a[n] != 0) return a[n];

        // 2. 动态规划填表
        // 外层循环 i：代表当前正在计算拥有 i 个节点的 BST 总数
        for (int i = 2; i <= n; i ++)
        {
            // 内层循环 j：尝试让第 j 个数字作为根节点
            // 数字范围是 [1, i]
            for (int j = 1; j <= i; j ++)
            {
                // 核心状态转移方程：
                // 左子树节点数：j - 1
                // 右子树节点数：i - j
                // 累加所有以不同 j 为根的情况
                a[i] += a[j - 1] * a[i - j];
            }
        }
        
        return a[n];
    }
};
```
