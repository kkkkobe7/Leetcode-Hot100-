# 581. 最短无序连续子数组

解题思路拆解

**1. 寻找初步的“混乱边界”**
*   我们从左往右看，找到第一个开始“下跌”的地方（`l`）。
*   我们从右往左看，找到第一个开始“上升”的地方（`r`）。
*   如果 `l >= r`，说明数组本来就是升序的，直接返回 0。

**2. 深度剖析：混乱区间的本质**
*   仅仅找到 `l` 和 `r` 是不够的。例如 `[1, 3, 2, 4]`，初步看混乱是 `[3, 2]`。
*   但如果数组是 `[2, 6, 4, 8, 10, 9, 15]`：
    *   左边下跌点在 `6`（索引 1），右边上升点在 `9`（索引 5）。
    *   在这个初步区间 `[6, 4, 8, 10, 9]` 中，最小值是 `4`，最大值是 `10`。
*   **关键点**：要让排序后的中间段能和左右两边完美衔接，必须保证：
    *   **左边有序段的所有数** 都必须小于等于 **混乱段的最小值**。
    *   **右边有序段的所有数** 都必须大于等于 **混乱段的最大值**。

**3. 最终扩张边界**
*   我们要拿着刚才找到的“中间段最小值”，往左扫描，看看左边有没有人比它大？如果有，那个人也得被卷入重新排序的范围（`l` 向左移）。
*   同理，拿着“中间段最大值”往右扫描，看看右边有没有人比它小？如果有，`r` 向右移。

简洁版代码：

```c++
class Solution {
public:
    int findUnsortedSubarray(vector<int>& nums) {
        int n = nums.size(), l = 0, r = n - 1;

        // 找到不满足升序的子数组
        while (l < n - 1 && nums[l] <= nums[l + 1]) l ++;
        while (r > 0 && nums[r] >= nums[r - 1]) r --;
        if (l >= r) return 0;

        // 找到其中的最大最小值
        int mi = 1e5+10, ma = -1e5-10;
        for (int i = l; i <= r; i ++)
        {
            mi = min(mi, nums[i]);
            ma = max(ma, nums[i]);
        }

        // 通过最大最小值来确定左右边界
        while (r < n && nums[r] < ma) r ++;
        while (l >= 0 && nums[l] > mi) l --;
        return r - l - 1;
    }
};
```

注释版代码：

```c++
class Solution {
public:
    int findUnsortedSubarray(vector<int>& nums) {
        int n = nums.size();
        if (n <= 1) return 0;

        int l = 0, r = n - 1;

        // 1. 从左向右寻找第一个破坏升序的位置
        // [2, 6, 4, ...] -> 到了 6，发现后面是 4，停止，此时 l=1
        while (l < n - 1 && nums[l] <= nums[l + 1]) l ++;
        
        // 2. 从右向左寻找第一个破坏降序（即逆向升序）的位置
        // [..., 10, 9, 15] -> 到了 9，发现前面是 10，停止，此时 r=5
        while (r > 0 && nums[r] >= nums[r - 1]) r --;

        // 如果 l 和 r 碰头甚至错开了，说明整个数组已经是有序的
        if (l >= r) return 0;

        // 3. 找到当前 [l, r] 范围内的绝对最大值和最小值
        // 这一步非常关键，因为中间段的任何一个数都可能影响最终边界
        int mi = 1e5+10, ma = -1e5-10;
        for (int i = l; i <= r; i ++)
        {
            mi = min(mi, nums[i]);
            ma = max(ma, nums[i]);
        }

        // 4. 扩张边界
        // 如果左边的有序部分里，有数比中间段的最小值 mi 还要大
        // 那么那个数也必须包含在重排范围内，l 往左退
        while (l >= 0 && nums[l] > mi) l --;
        
        // 如果右边的有序部分里，有数比中间段的最大值 ma 还要小
        // 那么那个数也必须包含在重排范围内，r 往右进
        while (r < n && nums[r] < ma) r ++;

        // 5. 计算长度
        // 此时 l 指向的是左侧第一个满足条件的数，r 指向的是右侧第一个满足条件的数
        // 实际无序区间是 (l, r)，不包含 l 和 r
        // 长度公式：(r - 1) - (l + 1) + 1 = r - l - 1
        return r - l - 1;
    }
};
```
