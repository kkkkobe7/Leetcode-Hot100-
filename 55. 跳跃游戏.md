# 55. 跳跃游戏

解题思路拆解

**1. 核心模型：领地扩张**
*   想象你在探索一片荒岛，你在每一个位置 `i` 拿到的数字 `nums[i]` 就是你的“燃料”，能让你从当前位置再往后探索 `nums[i]` 的距离。
*   我们维护一个变量 `max_reach`，代表当前我们能够到达的**最远边界**。

**2. 贪心策略：动态更新边界**
*   我们从左到右遍历数组。对于每一个位置 `i`：
    *   **首先检查**：当前位置 `i` 是否在我们的“领地”内？即 `i <= max_reach`。如果 `i > max_reach`，说明我们永远无法走到这一步，任务失败。
    *   **尝试扩张**：如果我们在领地内，就看看从当前位置出发，能不能把领地扩张得更远。新的可能边界是 `i + nums[i]`。
    *   **更新公式**：`max_reach = max(max_reach, i + nums[i])`。

**3. 提前结束（剪枝）**
*   在遍历的过程中，一旦发现 `max_reach` 已经超过或等于了最后一个下标，就说明我们一定能跳到终点，可以直接返回 `true`。

注释版代码：

```c++
class Solution {
public:
    bool canJump(vector<int>& nums) {
        // 1. max_reach 记录目前能够到达的最远位置下标
        int max_reach = 0; 
        int n = nums.size();

        for (int i = 0; i < n; i++) {
            // 2. 核心判断：如果当前位置 i 已经超过了目前能到达的最远距离
            // 说明没有任何一种跳法能跳到 i 这个位置，更别提终点了
            if (i > max_reach) return false; 
            
            // 3. 贪心更新：在当前位置尝试跳跃，看看能不能刷新最远距离
            // 当前位置 i + 该位置能跳的最大长度 nums[i]
            max_reach = max(max_reach, i + nums[i]);
            
            // 4. 优化：如果最远距离已经覆盖了最后一个位置，提前返回成功
            if (max_reach >= n - 1) return true;
        }

        // 5. 如果能完整跑完循环，说明每个位置都在可达范围内
        return true;
    }
};
```
