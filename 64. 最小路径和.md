# 64. 最小路径和

解题思路拆解

**1. 核心洞察：从哪里来？**
*   题目规定：每次只能**向下**或者**向右**移动。
*   反过来想：如果你现在站在格子 `(i, j)`，你只能是从两个地方走过来的：
    1.  从**上方**走过来，即 `(i-1, j)`。
    2.  从**左方**走过来，即 `(i, j-1)`。

**2. 定义状态 (State Definition)**
*   我们定义 `dp[i][j]` 为：从左上角 `(0, 0)` 出发，到达格子 `(i, j)` 的最小路径总和。
*   我们的最终目标是求 `dp[n-1][m-1]`。

**3. 状态转移方程 (Transition)**
*   既然只有两条路能到 `(i, j)`，那为了总和最小，我们肯定选之前路径和更小的那条：
*   **公式**：`dp[i][j] = min(dp[i-1][j], dp[i][j-1]) + grid[i][j]`

**4. 边界处理 (Initialization)**
*   **起点**：`dp[0][0] = grid[0][0]`。
*   **第一行**：只能一直向右走，所以 `dp[0][j] = dp[0][j-1] + grid[0][j]`。
*   **第一列**：只能一直向下走，所以 `dp[i][0] = dp[i-1][0] + grid[i][0]`。



简洁版代码：

```c++
class Solution {
public:
    static const int N = 210;
    int dp[N][N];

    int minPathSum(vector<vector<int>>& grid) {
        int n = grid.size(), m = grid[0].size();

        dp[0][0] = grid[0][0];
        for (int i = 1; i < n; i ++) dp[i][0] = dp[i-1][0] + grid[i][0];
        for (int j = 1; j < m; j ++) dp[0][j] = dp[0][j-1] + grid[0][j];

        for (int i = 1; i < n; i ++)
        {
            for (int j = 1; j < m; j ++)
            {
                dp[i][j] = min(dp[i-1][j], dp[i][j-1]) + grid[i][j];
            }
        }
        return dp[n-1][m-1];
    }
};
```

注释版代码：

```c++
class Solution {
public:
    // 假设最大网格为 200x200，N=210 是安全的
    static const int N = 210;
    int dp[N][N];

    int minPathSum(vector<vector<int>>& grid) {
        int n = grid.size();    // 行数
        int m = grid[0].size(); // 列数

        // 1. 初始化起点
        dp[0][0] = grid[0][0];

        // 2. 初始化第一列 (只能从上方走下来)
        for (int i = 1; i < n; i++) {
            dp[i][0] = dp[i-1][0] + grid[i][0];
        }

        // 3. 初始化第一行 (只能从左方走过来)
        for (int j = 1; j < m; j++) {
            dp[0][j] = dp[0][j-1] + grid[0][j];
        }

        // 4. 动态规划填表 (处理剩下的中间区域)
        for (int i = 1; i < n; i++)
        {
            for (int j = 1; j < m; j++)
            {
                // 核心状态转移：
                // 当前位置的最小和 = min(上方路径和, 左方路径和) + 当前格子本身的值
                dp[i][j] = min(dp[i-1][j], dp[i][j-1]) + grid[i][j];
            }
        }

        // 5. 返回右下角的结果
        return dp[n-1][m-1];
    }
};
```
