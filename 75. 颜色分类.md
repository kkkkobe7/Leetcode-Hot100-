# 75. 颜色分类

解题思路拆解

**1. 核心模型：三路划分 (3-Way Partitioning)**
*   我们将数组想象成三个待填充的区间：
    *   **0 区域**：存放红色（最左侧）。
    *   **1 区域**：存放白色（中间）。
    *   **2 区域**：存放蓝色（最右侧）。
*   我们的目标是维护三个“边界指针”，告诉我们下一个 0、1、2 应该放在哪里。

**2. 理解你的“边界扩张”逻辑（代码亮点）**
你提供的代码采用了一种非常独特的“推赶”式逻辑：
*   **id1, id2, id3** 分别代表：**0 应该放置的位置**、**1 应该放置的位置**、**2 应该放置的位置**。
*   这三个指针是**联动**的：
    *   当你发现一个 `0` 并把它交换到 `id1` 时，原本在 `id1` 位置的数（可能是 1 或 2）被挤到了后面。
    *   为了保持逻辑正确，`id2` 和 `id3` 也必须相应地向后挪动，确保 `id1 <= id2 <= id3`。
*   这种思路像是在推一个三层的屏障，每发现一个更小的数，就把所有屏障整体向右推。

**3. 为什么这种逻辑有效？**
*   当你交换 `nums[i]` 和 `nums[id1]` 时，你其实是把一个 `0` 归位了。
*   因为你维护了 `id1 <= id2 <= id3`，这就保证了：
    *   `[0, id1)` 全是 0。
    *   `[id1, id2)` 全是 1。
    *   `[id2, id3)` 全是 2。



简洁版代码：

```c++
class Solution {
public:
    void sortColors(vector<int>& nums) {
        int id1 = 0, id2 = 0, id3 = 0, n = nums.size(), i = 0;
        while (i < n)
        {
            if (nums[i] == 0)
            {
                if (i != id1) swap(nums[i], nums[id1]);
                else i ++;
                id1 ++;
                if (id1 > id2) id2 ++;
                if (id2 > id3) id3 ++;
            }
            else if (nums[i] == 1)
            {
                if (i != id2) swap(nums[i], nums[id2]);
                else i ++;
                id2 ++;
                if (id2 > id3) id3 ++;
            }
            else
            {
                if (i != id3) swap(nums[i], nums[id3]);
                else i ++;
                id3 ++;
            }   
        }

        return ;
    }
};
```

注释版代码：

```c++
class Solution {
public:
    void sortColors(vector<int>& nums) {
        // id1: 下一个 0 应该放的位置
        // id2: 下一个 1 应该放的位置
        // id3: 下一个 2 应该放的位置
        int id1 = 0, id2 = 0, id3 = 0;
        int n = nums.size();
        int i = 0; // i 是探索指针

        while (i < n)
        {
            // 情况 1: 发现红色 (0)
            if (nums[i] == 0)
            {
                // 如果当前位置不是 0 的边界，则交换
                if (i != id1) swap(nums[i], nums[id1]);
                else i ++; // 如果刚好重合，探索指针前进
                
                id1 ++; // 0 区域扩大
                // 联动：1 区域和 2 区域的起始点不能在 0 区域内部
                if (id1 > id2) id2 ++;
                if (id2 > id3) id3 ++;
            }
            // 情况 2: 发现白色 (1)
            else if (nums[i] == 1)
            {
                // 如果当前位置不是 1 的边界，则交换
                if (i != id2) swap(nums[i], nums[id2]);
                else i ++;
                
                id2 ++; // 1 区域扩大
                // 联动：2 区域的起始点不能在 1 区域内部
                if (id2 > id3) id3 ++;
            }
            // 情况 3: 发现蓝色 (2)
            else
            {
                // 如果当前位置不是 2 的边界，则交换
                if (i != id3) swap(nums[i], nums[id3]);
                else i ++;
                
                id3 ++; // 2 区域扩大
            }   
        }
    }
};
```
