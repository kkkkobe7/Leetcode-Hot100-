# 76. 最小覆盖子串

解题思路拆解

**1. 核心模型：两个“记事本”**
*   我们需要两个频率表：
    *   `hash1`：记录目标字符串 `t` 中每个字符需要的数量（“任务清单”）。
    *   `hash2`：记录当前窗口内每个字符已经搜集到的数量（“购物篮”）。
*   **变量 `diff`**：记录还有多少**种类**的字符没有凑齐。比如 `t` 是 "AABC"，那么字符种类就是 'A', 'B', 'C' 这 3 种。

**2. 算法流程：先扩张，再收缩**
*   **第一步：扩张 (右指针 `r` 移动)**
    *   不断增加右边界 `r`，把字符加入 `hash2`。
    *   如果某个字符搜集的数量**刚好达到**了 `hash1` 要求的数量，说明这个字符的任务完成了，`diff--`。
*   **第二步：收缩 (左指针 `l` 移动)**
    *   当 `diff == 0` 时，说明当前窗口已经覆盖了 `t` 的所有要求。
    *   **关键点**：此时窗口可能很长，我们要尝试把左边的字符扔掉。
    *   如果左边的字符 `s[l]` 不在任务清单里，或者 `hash2` 里的数量多于 `hash1` 里的数量（即它是多余的），我们就把 `l` 往右移。
*   **第三步：记录结果**
    *   在窗口满足条件且尽可能缩短后，记录此时的长度。如果是史上最短，更新答案。

**3. 形象化理解：搜集食材**
想象你在一条很长的走廊（字符串 `s`）里走，要按照食谱（字符串 `t`）搜集食材。
1.  你推着购物车往右走，只要还没凑齐食谱上的东西，你就一直往篮子里扔。
2.  一旦凑齐了（`diff == 0`），你停下来。
3.  你看一眼篮子最顶上的东西，如果是食谱不需要的，或者后面篮子里已经有很多重复的了，你就把它扔回货架（`l++`），直到不能再扔为止。
4.  量一下这段距离，记在心里。然后继续往右走，重复这个过程。



简洁版代码：

```c++
class Solution {
public:
    static const int N = 52;
    int hash1[N], hash2[N];

    int idx(char c)
    {
        if (c <= 'Z') return c - 'A';
        return 26 + c - 'a';
    }

    string minWindow(string s, string t) {
        int n = s.size(), m = t.size();
        if (n < m) return "";

        int diff = 0; // 有多少种不相同的字符
        for (auto &c: t)
        {
            int id = idx(c);
            if (hash1[id] == 0) diff ++;
            hash1[id] ++;
        }

        int l = 0, r = 0;
        string ans = "";
        while (r < n) // 窗口向右扩展直至满足diff == 0
        {
            int id = idx(s[r]);
            hash2[id] ++;
            if (hash1[id] && hash2[id] == hash1[id]) diff --; // 刚好匹配有效字符
            r ++;

            if (diff == 0)
            {
                while (l < r && hash1[idx(s[l])] < hash2[idx(s[l])]) // 删除多余字母
                {
                    hash2[idx(s[l])] --;
                    l ++; 
                }
                if (ans == "" || r - l < ans.size()) ans = s.substr(l, r - l);
            }
        } 

        return ans;
    }
};
```

注释版代码：

```c++
class Solution {
public:
    // 将 'A'-'Z' 和 'a'-'z' 映射到 0-51 的数组索引中
    static const int N = 52;
    int hash1[N], hash2[N];

    int idx(char c)
    {
        if (c <= 'Z') return c - 'A';
        return 26 + c - 'a';
    }

    string minWindow(string s, string t) {
        int n = s.size(), m = t.size();
        if (n < m) return "";

        // 1. 初始化任务清单 hash1
        int diff = 0; 
        for (auto &c: t)
        {
            int id = idx(c);
            if (hash1[id] == 0) diff ++; // 记录有多少种不同的字符需要匹配
            hash1[id] ++;
        }

        int l = 0, r = 0;
        string ans = "";
        
        // 2. 开始滑动窗口
        while (r < n) 
        {
            // 扩张右边界
            int id = idx(s[r]);
            hash2[id] ++;
            
            // 如果搜集到的数量刚好等于要求的数量，该种字符任务完成
            if (hash1[id] && hash2[id] == hash1[id]) diff --; 
            r ++;

            // 3. 当所有种类字符都凑齐了，尝试收缩左边界
            if (diff == 0)
            {
                // 如果左侧字符 s[l] 是多余的：
                // 情况 A: 任务压根没这个字符 (hash1[id] == 0)
                // 情况 B: 任务里有，但我们搜集多了 (hash2[id] > hash1[id])
                while (l < r && hash1[idx(s[l])] < hash2[idx(s[l])]) 
                {
                    hash2[idx(s[l])] --;
                    l ++; 
                }
                
                // 4. 更新最小窗口答案
                if (ans == "" || r - l < ans.size()) {
                    ans = s.substr(l, r - l);
                }
            }
        } 

        return ans;
    }
};
```
