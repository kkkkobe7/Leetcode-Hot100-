# 56. 合并区间

解题思路拆解

**1. 排序是核心前提**
*   **如果不排序**：你很难判断 `[1, 10]` 是否和 `[2, 3]` 或 `[9, 12]` 重叠，除非你把每一个区间都和剩下的所有区间比一遍，这太慢了。
*   **排序后**：我们按照区间的 **左端点（Start）** 升序排列。这样，如果两个区间能够合并，它们在数组里一定是**相邻**的。

**2. 核心逻辑：维护一个“当前正在合并”的区间**
*   我们设定两个变量 `ll` 和 `rr`，代表当前正在维护的合并区间的起点和终点。
*   **遍历数组中的每一个新区间 `[l, r]`**：
    *   **情况 A（有重叠）**：如果新区间起点 `l` 小于等于当前区间的终点 `rr`。
        *   说明两个区间连上了。我们需要更新 `rr`。
        *   **注意**：`rr` 应该取两者中更大的那个，即 `rr = max(rr, r)`。因为新区间可能完全被老区间包围。
    *   **情况 B（无重叠）**：如果新区间起点 `l` 大于当前区间的终点 `rr`。
        *   说明中间断开了，刚才那个合并区间已经彻底完成了。
        *   把 `[ll, rr]` 加入答案数组。
        *   然后，把当前这个新区间 `[l, r]` 作为“新的合并起点”，继续向后看。

**3. 收尾工作**
*   循环结束时，最后一个正在维护的区间还没有被存入结果中，别忘了把它 `push_back` 进去。



简洁版代码：

```c++
class Solution {
public:
    static bool cmp(const vector<int>& a, const vector<int>& b)
    {
        if (a[0] < b[0]) return true;
        if (a[0] > b[0]) return false;
        return a[1] < b[1];
    }

    vector<vector<int>> merge(vector<vector<int>>& intervals) {
        if (intervals.size() == 1) return intervals;
        
        vector<vector<int>> ans;
        sort(intervals.begin(), intervals.end(), cmp);

        int ll = intervals[0][0], rr = intervals[0][1], n = intervals.size();
        for (int i = 1; i < n; i ++)
        {
            int l = intervals[i][0], r = intervals[i][1];
            if (l > rr)
            {
                ans.push_back({ll, rr});
                ll = l;
                rr = r;
            }
            else
            {
                rr = max(rr, r);
            }
        }
        ans.push_back({ll, rr});
        return ans;
    }
};
```

注释版代码：

```c++
class Solution {
public:
    // 1. 自定义比较器：按左端点升序排列
    static bool cmp(const vector<int>& a, const vector<int>& b)
    {
        if (a[0] < b[0]) return true;
        if (a[0] > b[0]) return false;
        return a[1] < b[1]; // 如果左端点相同，按右端点升序
    }

    vector<vector<int>> merge(vector<vector<int>>& intervals) {
        // 特判：如果只有一个区间，直接返回
        if (intervals.size() <= 1) return intervals;
        
        vector<vector<int>> ans;
        // 2. 排序：这是贪心策略的基础
        sort(intervals.begin(), intervals.end(), cmp);

        // 3. 初始化第一个区间作为“合并基准”
        int ll = intervals[0][0], rr = intervals[0][1];
        int n = intervals.size();

        for (int i = 1; i < n; i ++)
        {
            int l = intervals[i][0], r = intervals[i][1];
            
            // 4. 核心逻辑判断
            if (l > rr)
            {
                // 情况 B：当前区间与之前的合并区间不连续（有空隙）
                ans.push_back({ll, rr}); // 结算上一个区间
                ll = l; // 开启新的合并区间
                rr = r;
            }
            else
            {
                // 情况 A：当前区间与之前的合并区间有重叠
                // 我们只需要更新右边界，取最大值以覆盖更多范围
                rr = max(rr, r);
            }
        }
        
        // 5. 循环结束，别忘了放入最后一个正在维护的区间
        ans.push_back({ll, rr});
        
        return ans;
    }
};
```
