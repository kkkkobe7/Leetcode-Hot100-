# 84. 柱状图中最大的矩形

解题思路拆解

**1. 核心直觉：以谁为准？**
*   与其漫无目的地组合柱子，不如问一个核心问题：**如果我必须以第 $i$ 根柱子的高度作为矩形的高度，这个矩形最宽能有多宽？**
*   **规则**：只要两边的柱子比我高（或一样高），我就可以向两边扩张。一旦遇到比我矮的柱子，扩张就停止。
*   **结论**：我们要找的是第 $i$ 根柱子左边第一个比它矮的（$L_i$）和右边第一个比它矮的（$R_i$）。
*   **面积公式**：$Area = heights[i] \times (R_i - L_i - 1)$。

**2. 为什么需要单调栈？**
*   我们需要高效地寻找“下一个更小”和“上一个更小”的元素。
*   单调栈就像一个“排队系统”。当我们遇到一根新柱子 $i$ 时，如果它比栈顶的柱子矮，那么对于栈顶那根柱子来说，**右边的边界终于找到了**！

**3. 算法推导过程（单调递增栈）**
*   我们维护一个存储下标的栈，要求栈内下标对应的柱子高度是**递增**的。
*   **遍历每一根柱子 $i$**：
    *   **当当前柱子 $i$ 的高度 < 栈顶高度时**：说明栈顶柱子找到了它的右边界 $R$（即当前的 $i$）。我们把栈顶弹出，并记录它的 $R$。
    *   **由于栈是递增的**：弹出的柱子，其左边界 $L$ 恰好就是它在栈中的“前任”（即新的栈顶）。
*   **边界处理**：
    *   如果左边没有更矮的，左边界设为 $-1$。
    *   如果右边没有更矮的，右边界设为 $n$（数组末尾）。

简洁版代码：

``` c++
class Solution {
public:
    /*
    找到每一个数左侧和右侧第一个小于它的
    */
    int largestRectangleArea(vector<int>& heights) {
        int n = heights.size(), ans = 0;
        vector<int> l(n), r(n, n);
        
        stack<int> st;  // 递增
        for (int i = 0; i < n; i ++)
        {
            while (st.size() && heights[st.top()] >= heights[i])  // 维护递增栈
            {
                r[st.top()] = i; // 对于下标st.top()来说，右侧第一个小于其的位置为i
                st.pop();
            }
            l[i] = st.empty()? -1: st.top(); // 对于位置i来说，左侧第一个小于其的位置为st.top()
            st.push(i);
        }

        for (int i = 0; i < n; i ++) ans = max(ans, heights[i]*(r[i]-l[i]-1));
        return ans;
    }
};
```

注释版代码：

```c++
class Solution {
public:
    int largestRectangleArea(vector<int>& heights) {
        int n = heights.size();
        int ans = 0;
        
        // l[i]: 存储第 i 根柱子左侧第一个比它矮的柱子下标
        // r[i]: 存储第 i 根柱子右侧第一个比它矮的柱子下标，预设为 n（右边界之外）
        vector<int> l(n), r(n, n);
        
        // 单调递增栈：栈底到栈顶对应的高度越来越大
        stack<int> st;  

        for (int i = 0; i < n; i ++)
        {
            // 1. 如果当前柱子比栈顶柱子矮
            // 说明栈顶柱子找到了它右边的“截止点”
            while (!st.empty() && heights[st.top()] >= heights[i])
            {
                // 对于被弹出的元素来说，i 就是右侧第一个比它小的位置
                r[st.top()] = i; 
                st.pop();
            }
            
            // 2. 确定当前柱子的左边界
            // 由于栈是递增的，当前元素入栈前，栈顶就是它左侧第一个比它矮的元素
            // 如果栈为空，说明左边全是高个子，边界设为 -1
            l[i] = st.empty() ? -1 : st.top();
            
            // 3. 将当前柱子下标入栈
            st.push(i);
        }

        // 4. 结算：遍历每一根柱子，计算以它为高的矩形面积
        for (int i = 0; i < n; i ++) 
        {
            // 宽度计算公式：右边界 - 左边界 - 1
            // 比如：左边第一个小的是 -1，右边第一个小的是 2，则宽度是 2 - (-1) - 1 = 2 (包含下标 0, 1)
            ans = max(ans, heights[i] * (r[i] - l[i] - 1));
        }
        
        return ans;
    }
};
```
