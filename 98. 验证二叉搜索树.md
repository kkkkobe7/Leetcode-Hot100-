# 98. 验证二叉搜索树

解题思路拆解

**1. 核心洞察：中序遍历的有序性**
*   二叉搜索树有一个非常重要的物理特性：如果我们按照“左-根-右”的顺序遍历它，得到的数字序列应该是**从小到大排列**的。
*   **推论**：在遍历过程中，**当前访问的节点值，必须大于上一个刚访问过的节点值。**

**2. 算法设计：带状态的递归**
*   我们需要一个变量（代码中的 `la`）来记录“上一个刚刚访问过的节点”。
*   当访问当前节点 `p` 时：
    1.  检查 `la` 是否为空（如果为空，说明 `p` 是整棵树最小的那个起点）。
    2.  如果 `la` 不为空，检查是否满足 `p->val > la->val`。
    3.  如果不满足（即 `p->val <= la->val`），说明这不是一棵 BST，立即标记失败。
    4.  更新 `la = p`，继续向右走。

**3. 细节技巧：引用传递（Pointer Reference）**
*   在递归函数中，`TreeNode* &la` 使用了**指针的引用**。
*   **为什么要用引用？** 因为递归函数在处理左子树时会更新 `la` 的值，我们希望回到根节点处理时，拿到的 `la` 是左子树里最后（也是最大）的那个节点。引用保证了 `la` 这个“笔记本”在所有递归层级中是**同步更新**的。

简洁版代码：

```c++
class Solution {
public:

    // 二叉搜索树 左 < 中 < 右，要保证左边最大的都小于中间，右边最小的都大于中间
    // 中序遍历，左中右。
    bool ans = true;
    void in(TreeNode* p, TreeNode* &la)
    {
        if (!ans) return ;
        if (p == nullptr) return ;

        in(p->left, la);
        if (la != nullptr)
        {
            if (p->val <= la->val)
            {
                ans = false;
                return ;
            }
        }
        la = p;
        in(p->right, la);
    }

    bool isValidBST(TreeNode* root) {
        TreeNode* p = nullptr;
        in(root, p);
        return ans;
    }
};
```

注释版代码：

```c++
class Solution {
public:
    // ans: 全局结果标记，一旦发现不合法则置为 false
    bool ans = true;

    // in 函数：中序遍历函数
    // p: 当前访问的节点
    // la: 上一个访问过的节点的引用（核心：同步更新状态）
    void in(TreeNode* p, TreeNode* &la)
    {
        // 1. 剪枝：如果已经发现不是 BST 了，直接返回，不再浪费计算资源
        if (!ans) return;
        
        // 2. 递归终止：走到叶子节点外
        if (p == nullptr) return;

        // 3. 【左】：递归访问左子树
        in(p->left, la);

        // 4. 【中】：处理当前根节点逻辑
        // 如果 la（上一个节点）存在，且当前值不大于上一个值
        if (la != nullptr)
        {
            if (p->val <= la->val)
            {
                ans = false; // 标记失败：不符合严格递增
                return;
            }
        }
        // 更新 la 为当前节点，供下一个节点对比
        la = p;

        // 5. 【右】：递归访问右子树
        in(p->right, la);
    }

    bool isValidBST(TreeNode* root) {
        // p 指针初始为空，代表还没访问过任何节点
        TreeNode* p = nullptr;
        // 重置状态
        ans = true;
        
        in(root, p);
        return ans;
    }
};
```
