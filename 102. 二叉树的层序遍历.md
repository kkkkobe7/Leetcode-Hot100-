# 102. 二叉树的层序遍历

解题思路拆解

**1. 核心工具：队列 (Queue)**
*   层序遍历的本质是先遇到的节点先处理，后遇到的节点后处理。
*   **先进先出 (FIFO)** 的特性正好符合我们的需求，所以我们选择**队列**作为辅助数据结构。

**2. 核心挑战：如何区分“层”？**
*   普通的 BFS 只是把所有节点按顺序吐出来，但题目要求按层返回（一个二维数组）。
*   **解决方案：** 在处理每一层之前，先记录当前队列的大小 `n = q.size()`。
    *   这个 `n` 就代表了**当前这一层一共有多少个节点**。
    *   我们连续弹出这 `n` 个节点，并将它们产生的子节点加入队列，这样就能保证这次循环处理的全部都是同一层的节点。

**3. 算法流程**
1.  **入队**：将根节点放入队列。
2.  **外层循环**：只要队列不为空，说明还有楼层没爬完。
3.  **内层循环**：
    *   获取当前层的节点数 `n`。
    *   循环 `n` 次：弹出节点 -> 记录值 -> 将其左右孩子（如果有）送入队列。
4.  **记录**：将这一层的结果存入最终答案数组。



简洁版代码：

```c++
class Solution {
public:
    vector<vector<int>> levelOrder(TreeNode* root) {
        if (root == nullptr) return {};
        vector<vector<int>> ans;
        queue<TreeNode*> q;
        q.push(root);
        while (q.size())
        {
            vector<int> cur;
            int n = q.size();
            while (n --)
            {
                TreeNode* t = q.front();
                q.pop();
                cur.push_back(t->val);
                if (t->left) q.push(t->left);
                if (t->right) q.push(t->right);
            }
            ans.push_back(cur);
        } 

        return ans;
    }
};
```

注释版代码：

```c++
class Solution {
public:
    vector<vector<int>> levelOrder(TreeNode* root) {
        // 1. 特判：如果根节点为空，直接返回空数组
        if (root == nullptr) return {};

        vector<vector<int>> ans; // 存储最终的二维结果
        queue<TreeNode*> q;      // 辅助队列
        
        // 2. 将起点（根节点）入队
        q.push(root);

        // 3. 开始 BFS 遍历
        while (!q.empty())
        {
            vector<int> cur;       // 存储当前层节点的值
            int n = q.size();      // 【关键】获取当前层的节点个数
            
            // 4. 连续处理当前层的所有节点
            while (n--)
            {
                // 弹出队首节点
                TreeNode* t = q.front();
                q.pop();
                
                // 记录当前节点的值
                cur.push_back(t->val);
                
                // 将下一层的节点（当前节点的左右孩子）放入队列
                if (t->left) q.push(t->left);
                if (t->right) q.push(t->right);
            }
            
            // 5. 当前层处理完毕，存入 ans
            ans.push_back(cur);
        } 

        return ans;
    }
};
```
