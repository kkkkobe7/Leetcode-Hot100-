# 78. 子集

解题思路拆解

**1. 核心模型：选与不选（二叉决策树）**
*   求数组的所有子集，本质上就是对数组中的**每一个元素**做一个决定：**“我是要把它放进子集里，还是不要放进子集里？”**
*   对于包含 $n$ 个元素的数组，我们需要做 $n$ 次决定。每次决定有两个分支（选 / 不选），所以最终会产生 $2^n$ 个叶子节点，这恰好对应 $2^n$ 个不同的子集。

**2. 递归函数的设计**
*   `dfs(nums, i)`：表示我们现在站在第 `i` 个元素面前，准备对它做决定。
*   **分支一（不选）**：什么都不做，直接走向下一个元素 `dfs(nums, i + 1)`。
*   **分支二（选）**：把 `nums[i]` 放进当前的“购物篮” `cur` 里，然后走向下一个元素 `dfs(nums, i + 1)`。
*   **回溯（关键）**：从“选”的分支退回来时，我们需要把刚才放进篮子里的 `nums[i]` **拿出来**，恢复现场，这样才不会影响后续其他的决策分支。

**3. 终止条件**
*   当 `i == nums.size()` 时，说明我们已经对所有的元素都做完了决定。此时“购物篮” `cur` 里的东西就是一个完整的子集，我们把它装进最终的答案 `ans` 里即可。



简洁版代码：

```c++
class Solution {
public:
    vector<int> cur;
    vector<vector<int>> ans;

    void dfs(vector<int>& nums, int i)
    {
        if (i == nums.size())
        {
            ans.push_back(cur);
            return ;
        }

        // 不选nums[i]
        dfs(nums, i + 1);

        // 选nums[i]
        cur.push_back(nums[i]);
        dfs(nums, i + 1);
        
        // 回溯
        cur.pop_back();
    }
    
    vector<vector<int>> subsets(vector<int>& nums) {
        dfs(nums, 0);
        return ans;
    }
};
```

注释版代码：

```c++
class Solution {
public:
    vector<int> cur;         // 记录当前子集（即“购物篮”）
    vector<vector<int>> ans; // 记录所有子集的集合

    // DFS 函数：当前正在决定是否将 nums[i] 加入子集
    void dfs(vector<int>& nums, int i)
    {
        // 1. 递归终止条件
        // 当 i 达到数组长度时，说明所有元素都遍历完了，形成了一个完整的子集
        if (i == nums.size())
        {
            ans.push_back(cur);
            return;
        }

        // 2. 左分支：不选当前的 nums[i]
        // 保持 cur 不变，直接去考虑下一个元素
        dfs(nums, i + 1);

        // 3. 右分支：选当前的 nums[i]
        // 将 nums[i] 加入当前子集
        cur.push_back(nums[i]);
        // 带着 nums[i] 继续考虑下一个元素
        dfs(nums, i + 1);
        
        // 4. 回溯：恢复现场
        // 当右分支探索完毕退回到这里时，必须把 nums[i] 从子集中移除
        // 以便后续（上一层）的递归不受影响
        cur.pop_back();
    }
    
    vector<vector<int>> subsets(vector<int>& nums) {
        // LeetCode 多次调用类实例时，需清空全局变量
        cur.clear();
        ans.clear();
        
        // 从第 0 个元素开始做决定
        dfs(nums, 0);
        
        return ans;
    }
};
```
